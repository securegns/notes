#### 🎯 Key Exam Points - Phase 5

| Topic | Must Remember |
|-------|---------------|
| ExpressRoute | Private connection, not over internet, requires provider |
| Private peering | Access VNets |
| Microsoft peering | Access Microsoft 365, Azure PaaS (public endpoints) |
| FastPath | Bypasses gateway for performance (10 Gbps+) |
| Virtual WAN Basic | VPN only; Standard = VPN + ExpressRoute + VNet |
